# Минус / Телега

Хакатонный веб-прототип на React + TypeScript + Vite + Framer Motion.

Сценарий:

1. Пользователь стартует в тяжёлом интерфейсе **Мин**.
2. Заходит в чат **«Жалобы на команду»**.
3. Проходит мини-игру модерации: если сообщение критикует команду — удалить, иначе оставить.
4. При достижении порога запускается плавный `morph_to_telega`.
5. В **Телеге** открывается вкладка **VPN** с clicker-механикой.
6. Если успеть заполнить круг — интерфейс остаётся в Телеге.
7. Если не успеть — стартует `morph_to_min` и система плавно деградирует обратно.

## Стек

- React 18
- TypeScript
- Vite
- Framer Motion
- Zustand
- только локальный state, без бэкенда и внешних API

## Запуск

```bash
npm install
npm run dev
```

Сборка production bundle:

```bash
npm run build
npm run preview
```

## Что внутри

- `src/app/store.ts` — централизованная state-machine
- `src/utils/morph.ts` — интерполяция дизайн-токенов через `getMorphTheme(progress, direction)`
- `src/features/moderation/*` — логика модерации
- `src/features/vpn/*` — логика VPN clicker
- `src/features/morph/*` — движок плавного перехода
- `src/components/min/*` — shell и UI для Мина
- `src/components/telega/*` — shell и UI для Телеги
- `src/components/hybrid/*` — промежуточный мутирующий интерфейс
- `src/components/shared/*` — общие shell-компоненты

## Замечания по дизайну

- **Мин** вдохновлён shell-подходом MAX UI: плотные панели, тяжёлый тёмный контейнер, крупные секции, search / folders / cell-like chat items.
- **Телега** построена как более лёгкий React-экран с mini-app настроением: светлый воздух, телеграмный ритм, мягкие bubbles, отдельная VPN-вкладка.
- Морфинг сделан через **интерполяцию токенов**: цвет, скругления, тени, отступы, nav-height, composer, bubbles, badge, фон и т.д.

## Демо-сценарий для показа жюри

1. Открыть приложение.
2. Показать плотный Мин.
3. Открыть **«Жалобы на команду»**.
4. Дать 6+ правильных решений подряд или набрать 10 точных решений.
5. Показать morph в Телегу.
6. Открыть **VPN**.
7. Быстро кликать до `Подключено`.
8. Подождать новую волну или специально провалить таймер.
9. Показать обратный morph в Мин.
